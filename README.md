# Crop Disease Classification using Deep Learning

This project implements a deep learning model for crop disease classification using **EfficientNet-B0 and PyTorch**.

## Features
- Transfer learning using EfficientNet-B0
- Stratified 5-Fold Cross Validation
- Custom PyTorch Dataset and DataLoader
- GPU training using Google Colab
- Evaluation using Precision, Recall, F1-score and Confusion Matrix

## Results
Average Cross-Validation Accuracy: **95.34%**

## Technologies Used
- Python
- PyTorch
- TIMM
- Scikit-learn
- Google Colab
- Matplotlib & Seaborn

## Dataset
Paddy Disease Classification Dataset (Kaggle)
